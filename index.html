Criminal Identification System

This Criminal Identification System consists of two modules,criminal sketching and identification application leveraging drag-and-drop features for creating accurate facial composite sketches. In the sketching module, users can manipulate various facial features such as eyes, nose, mouth, and hair, by dragging and dropping predefined elements to create composite sketches of suspects. These sketches are saved with relevant metadata for future reference. In the recognition module, users upload previously created sketches to compare them with a database of real criminal images. Utilizing facial recognition algorithms and machine learning techniques, the application matches the uploaded sketch with the most probable real criminal image from the database, providing potential leads for law enforcement agencies. By combining intuitive sketching tools with advanced recognition capabilities, this application offers a user-friendly interface for creating accurate sketches and aiding in criminal identification processes.

Depp Fake Detection in Videos

The "Deep Fake Detection in Videos" project aims to develop a robust system for identifying deepfake videos using a combination of convolutional neural networks (CNNs) and recurrent neural networks (RNNs). The project involves preprocessing video data, training a deep learning model, and deploying a user-friendly interface using Tkinter for video upload and real-time analysis. The system classifies videos as real or fake by analyzing extracted frames and leveraging a trained deepfake detection model. This project provides a comprehensive solution for detecting manipulated video content, contributing to the fight against misinformation and enhancing digital media integrity.

The dataset for this project comprises two categories of videos: Real Videos: Authentic videos without any manipulations and Fake Videos: Videos altered using deepfake techniques. The videos were preprocessed and divided into frames, which were then used for training the deep learning model.
